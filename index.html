計算式のやさしい解説

dS/dt = -β · S · I / N

→ 【未感染者の減少】：健康な人が感染者と出会って、新しく病気になるスピードです。マイナスがついているのは人数が減ることを意味します。

dI/dt = β · S · I / N - γ · I

→ 【感染者の増減】：新しく病気になる人から、治って元気になる人を引いた値です。この値がプラスなら流行は拡大し、マイナスなら収束します。

dR/dt = γ · I

→ 【回復者の増加】：病気が治って、免疫がついたり隔離されたりするスピードです。現在の感染者が多いほど、ここへ移動する人も増えます。
